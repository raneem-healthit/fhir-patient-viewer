# 🏥 MediView — FHIR Patient Explorer

Eine Web-App die echte Patientendaten vom FHIR R4 Test-Server abruft und anzeigt.

## Technologien
- Python
- Streamlit
- HL7 FHIR R4

## Features
- Echte FHIR-Patientendaten abrufen
- Nach Patienten suchen
- Geburtsdatum, Geschlecht, Adresse, Telefon anzeigen

## Projekt starten
pip3 install streamlit requests
python3.11 -m streamlit run app.py
